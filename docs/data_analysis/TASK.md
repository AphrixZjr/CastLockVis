# 数据分析任务

## 任务描述

**问题**：可视化系统已经开发完成，但这个系统所揭示的模式、规律或结论尚不清晰，`docs\overall_design\proposal.md` 中设想的“联动发现”案例也没有找到。

**任务**：你需要基于 `public\data` 中的清洗后数据及已实现的视图联动逻辑，开展探索性数据分析。你力图得出宏观和微观两方面的结论：宏观上，根据四张视图中数据的统计结果（聚类、香农熵增减、转型成败、马尔科夫矩阵），描述类型转移的整体规律；微观上，参照 proposal 的案例逻辑，试图找到几个可解释性强的“转型成功/失败”案例。

## 操作要求

在该文件夹（`docs\data_analysis`）下运行所需脚本。分析报告也写回这个文件夹中。

你不应该改动除此以外的内容，除非显式要求。因为你的任务并非数据处理和系统开发。
